## 项目概览

`oh-my-pet` 是一个早期公开项目，目标是在 macOS 上探索一个由用户真正拥有的 AI 桌面陪伴宠物。

它不是“套了可爱外壳的通用聊天机器人”。它更像一个安静、个人化、可以被创造出来的桌面小伙伴：有自己的形象、声音、记忆，以及和用户一起完成事情的小仪式。

我们希望它给人的感觉是：

> 这是我亲手创造的小伙伴。它住在我的桌面上，安静地陪着我，也记得我们一起完成过的小事。

## 当前状态

`oh-my-pet` 目前处于 **pre-alpha 产品设计阶段**。这个仓库里还没有可运行应用。

仓库当前主要包含公开的产品方向和协作说明。早期实现大概率会从小型 macOS 实验、本地宠物档案格式、轻量 UI 原型开始。

项目会尽量保持小步、高频、可见的提交节奏，让关注者更容易理解进展，也更容易参与 review。

## 核心想法

- **macOS 优先**：先把原生桌面体验做好。
- **情绪价值优先**：安静陪伴比不停说话更重要。
- **用户创造形象**：用户可以用 AI 生成宠物形象和声音。
- **本地宠物档案**：宠物素材、性格、声音信息、记忆和设置应保存在本地档案里。
- **支持自带服务密钥**：用户应能配置自己的 AI provider key。
- **权限透明**：宠物可以显得“懂场景”，但必须诚实说明自己能访问什么。

## 计划中的 MVP

第一阶段产品方向围绕五个系统展开。

| 模块 | 方向 |
| --- | --- |
| AI Pet Studio | 通过文生图、图生图、参考图、状态图、声音生成，以及基于明确同意的声音克隆来创建宠物。 |
| Desktop Pet Runtime | 让宠物以透明浮层形式出现在 macOS 桌面上，支持拖拽、缩放、隐藏/显示、置顶和安静的状态变化。 |
| Pet House | 保存宠物身份、视觉状态、声音档案、小记忆、贴纸、小房间物件、生成历史和导入/导出数据。 |
| Focus And Task Companion | 在专注和任务完成过程中安静陪伴用户，并把完成时刻记录成小的共同记忆。 |
| Scene-Based Selection Assistant | 只在用户主动触发时处理选中文本，并清楚说明可能发送给已配置 provider 的内容类型。 |

第一版运行时可以先使用透明静态状态图加轻量动画。宠物档案格式应为后续更丰富的形态留下空间，比如 sprite、Live2D、Spine、骨骼动画或视频 sprite。

## 隐私与权限

隐私是产品设计的一部分，不是后面再补的功能。

预期边界：

- 不偷偷看屏幕。
- 不偷偷监听麦克风。
- 不自动读取文件、浏览器内容、聊天记录或选中文本。
- MVP 不做持续屏幕观察。
- 选中文本只能在用户主动触发后读取。
- AI 动作应显示涉及的 provider、model 和数据类型。
- Provider key 应保存在 macOS Keychain。
- 生成的宠物素材、声音信息、记忆和设置应保存在本地。

App/window awareness 可能需要 macOS Accessibility 权限。预期用途应限制在当前 app 身份、窗口标题、基础窗口位置等上下文信息上，不应等同于持续读取屏幕。

## 技术方向

当前设计更倾向于原生 macOS 原型：

- 使用 SwiftUI + AppKit 构建宿主应用。
- 使用 AppKit 处理透明浮窗、菜单栏行为、窗口层级、拖拽/缩放和桌面集成。
- 使用 macOS Keychain 保存 provider key。
- 使用 Accessibility APIs 支持可选的 app/window awareness。
- 使用本地宠物档案包保存视觉素材、声音信息、行为映射、Pet House 数据和记忆记录。
- 为 AI 图像、声音和文本调用提供 provider adapter。

这仍然是设计方向，不代表已经实现的技术栈。

## 参与协作

这里是一个开放协作空间，欢迎关心桌面陪伴、用户本地控制、macOS 体验、隐私和诚实 AI 的朋友一起参与。

现阶段有价值的贡献包括：

- macOS 透明浮窗实验。
- 宠物档案格式反馈。
- AI 图像一致性实验。
- 声音生成与同意流程 UX 反馈。
- 隐私和权限 review。
- Pet Studio、Pet House、桌面宠物行为的小型 UI 原型。

小贡献非常欢迎。一个聚焦的 issue、一个短实验、一次细致的隐私 review、一个小型可运行原型，往往比大范围重写更有价值。

较大改动前，请先打开 issue 讨论，让项目方向保持一致。

## 暂不做的事

短期内不做：

- 治疗或医疗相关承诺。
- 完整的自主电脑控制。
- 持续屏幕录制。
- 未经同意读取文件、浏览器内容、聊天记录或选中文本。
- 未经同意的声音克隆。
- 饥饿衰减、健康惩罚、惩罚机制或愧疚循环。
- NFT 或链上资产。
- 未授权 IP 角色包。
- 强制账号系统。
